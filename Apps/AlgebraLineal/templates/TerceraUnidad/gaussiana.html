Eliminación Gaussiana

Definición:

El método de Gauss también se conoce como eliminación gaussiana. Este método es muy importante, ya que te permite resolver matrices de una manera sistemática y siguiendo un proceso paso por paso.

Pasos:

1. Llevar un sistema de ecuaciones a la forma matricial.
2. Convertir una matriz cuadrada en una matriz triangular superior, que es equivalente a la matriz original.
3. Resolver el sistema, sustituyendo las variables en cada ecuación resultante.

El objetivo de la eliminación gaussiana es llegar a la matriz triangular superior que es equivalente a la matriz original. Hallar las soluciones, a partir de una matriz triangular, es muy sencillo. Para llegar a la matriz triangular superior con la que obtenemos la solución del sistema de ecuaciones, el proceso que debemos seguir es:
1. Intercambiar ecuaciones.
2. Multiplicar una o varias ecuaciones por números distintos de 0.
3. Sumar o restar ecuaciones entre sí.
